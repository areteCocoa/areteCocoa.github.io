Socratweet

2015

 Tweet inspiring tweets by stealing content from Reddit


  Myself

 Objective-C


Socratweet pulls data from the /r/showerthoughts subreddit as a list and presents them on a table. Tapping on a table cell composes a tweet for your Twitter account, which can then be posted to your timeline.

Evolution Simulator

2012-2013

 A customizable simulation for evolution among species


  Myself for the 2013 Intel Northwest Science Expo


 Java


This Java application generates simulated worlds using user parameters, and adds different organisms of various species that, given enough time, will evolve to the different areas that they are spawned in. A longer version of the simulation might include a realistic world – one with no organisms at day 0, while a simple simulation would include 100s of organisms at day 0.


 Evolution Simulator is on Github
